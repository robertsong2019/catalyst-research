# Prompt Engineering Evolution
> 从 Prompt Engineering 到 Agentic Engineering 的演进

## 一句话定义
Prompt Engineering 正在演变为 Agentic Engineering：从精心设计提示词，到设计完整的 Agent 系统（工具、记忆、编排、安全）。

## 演进路径

```
Prompt Engineering (2023)
  → Chain-of-Thought / Few-shot
    → Tool Use / Function Calling (2024)
      → Agent Loops / ReAct (2025)
        → Agentic Engineering (2026)
```

### Prompt Engineering 时代
- 核心技能：写好提示词
- 局限：单轮交互，无状态

### Agentic Engineering 时代
- 核心技能：设计 Agent 系统
- 组成：工具 + 记忆 + 编排 + 安全 + 监控

## 关键转变

| 维度 | Prompt Engineering | Agentic Engineering |
|------|-------------------|---------------------|
| **关注点** | 提示词质量 | 系统设计 |
| **复杂度** | 单次调用 | 多步编排 |
| **状态** | 无状态 | 有状态 |
| **工具** | 不使用 | 核心能力 |
| **可靠性** | 靠提示词约束 | 靠 Guardrail + 测试 |
